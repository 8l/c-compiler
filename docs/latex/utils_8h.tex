Prints version information. 

This function uses the C time library to get the current local time. It then formats it in a buffer that holds the day of the build. This is then printed out. We first initialize a raw time variable of type t\_\-time. We also initialize a tm struct which holds component time parts. We feed the the raw time to a localtime function. Finally we use the strftime() to format the local to time a C string.
